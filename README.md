### 利用方法
- chrome-ty-asmr-playerの拡張を押すとポップアップが表示され、Chromeで開いているYouTubeの動画一覧が右に並びます。
- 動画を選択すると再生します。すでに再生中の場合は停止します。

### 実装予定
- 現在再生中の動画のステータスをわかりやすくする
- タブを閉じられるようにする
- 配信終了したら自動で閉じるなど
